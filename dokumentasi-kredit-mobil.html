📄 Kode Lengkap kalkulasi.php

PHP 
// Fungsi 1: Format angka jadi Rupiah
function formatRupiah(float $n): string {
    return 'Rp. ' . number_format((int) round($n), 0, ',', '.');
}

// Fungsi 2: Bersihkan input "250.000.000" → 250000000
function sanitasiAngka(string $s): float {
    return (float) preg_replace('/[^\d.]/', '', str_replace('.', '', $s));
}

// Fungsi 3: INTI PERHITUNGAN KREDIT
function hitungKredit(float $harga, float $dpPersen, int $tenorTahun): array {
    $bunga      = $harga * 0.20;
    $totalBunga = $harga + $bunga;
    $nilaiDp    = $harga * ($dpPersen / 100);
    $tenorBulan = $tenorTahun * 12;
    $sisaHutang = $totalBunga - $nilaiDp;
    $angsuran   = $sisaHutang / $tenorBulan;

    return compact(
        'harga', 'dpPersen', 'nilaiDp',
        'tenorTahun', 'tenorBulan',
        'bunga', 'angsuran'
    );
}
Algoritma

Algoritma Kredit Flat

Sistem ini menggunakan metode bunga flat (flat rate) — bunga dihitung dari harga asli mobil, bukan dari sisa hutang. Ini berbeda dari bunga efektif yang makin kecil setiap bulan.

🧮 Rumus Lengkap (Urutan Eksekusi)
Bunga Total = Harga Mobil × 20%
Total Hutang = Harga Mobil + Bunga Total
Nilai DP = Harga Mobil × (DP% ÷ 100)
Tenor Bulan = Tenor Tahun × 12
Sisa Hutang = Total Hutang − Nilai DP
Angsuran/Bulan = Sisa Hutang ÷ Tenor Bulan

🔢 Contoh Nyata

Skenario: Harga Mobil Rp 200.000.000 | DP 20% | Tenor 5 Tahun

Bunga = 200.000.000 × 20% = Rp 40.000.000
Total Hutang = 200.000.000 + 40.000.000 = Rp 240.000.000
Nilai DP = 200.000.000 × 20% = Rp 40.000.000
Tenor Bulan = 5 × 12 = 60 bulan
Sisa Hutang = 240.000.000 − 40.000.000 = Rp 200.000.000
Angsuran/Bulan = 200.000.000 ÷ 60 = Rp 3.333.333
⚠️ Catatan Bunga Flat: Bunga dihitung dari harga awal, bukan sisa pinjaman. Jadi angsuran setiap bulan selalu sama. Berbeda dengan KPR yang bunganya dari outstanding balance.

Validasi Input (PHP)

Sebelum menghitung, semua input divalidasi di sisi server. Ini mencegah error kalau user memasukkan data yang tidak masuk akal.

PHP — Blok POST handler 
// Cek apakah form dikirim
$posted = ($_SERVER['REQUEST_METHOD'] === 'POST'
            && ($_POST['action'] ?? '') === 'hitung');

if ($posted) {
    // Ambil & bersihkan input
    $harga      = sanitasiAngka($_POST['harga_mobil'] ?? '0');
    $dpPersen   = (float) ($_POST['dp']    ?? 0);
    $tenorTahun = (int)   ($_POST['tenor'] ?? 0);

    // Validasi: kumpulkan error
    if ($harga <= 0)                   $errors['harga']  = 'Harga tidak valid.';
    if ($dpPersen < 0 || $dpPersen > 99) $errors['dp']     = 'DP harus 0–99%.';
    if ($tenorTahun < 1 || > 10)       $errors['tenor']  = 'Tenor 1–10 tahun.';

    // Kalau tidak ada error, hitung!
    if (empty($errors)) {
        $r = hitungKredit($harga, $dpPersen, $tenorTahun);
        $result = [ ... ]; // simpan hasil + format rupiah
    }
}

Objek Kalkulator (JS)

Kalkulator JS memakai Object Literal — semua fungsi dibungkus dalam satu objek Kalkulator. Rumusnya identik 100% dengan PHP agar hasilnya sama.

JavaScript — Object Kalkulator 
const Kalkulator = {

  // Method 1: Validasi semua field
  validasi() {
    let ok = true;
    const harga = document.getElementById('hargaMobil');
    const dp    = document.getElementById('dp');
    const tenor = document.getElementById('tenor');

    // Hapus class error dulu
    [harga, dp, tenor].forEach(el => el.classList.remove('is-invalid'));

    // Validasi harga
    if (parseRupiah(harga.value) <= 0) {
      harga.classList.add('is-invalid'); ok = false;
    }
    return ok;
  },

  // Method 2: INTI HITUNG (sama dengan PHP)
  hitung() {
    if (!this.validasi()) return; // stop jika invalid

    const hargaMobil = parseRupiah(...);
    const dpPersen   = parseFloat(...);
    const tenorTahun = parseInt(...);

    // Rumus identik dengan PHP
    const bunga      = hargaMobil * 0.20;
    const totalBunga = hargaMobil + bunga;
    const nilaiDp    = hargaMobil * (dpPersen / 100);
    const tenorBulan = tenorTahun * 12;
    const sisaHutang = totalBunga - nilaiDp;
    const angsuran   = sisaHutang / tenorBulan;

    this.tampilkan({ hargaMobil, dpPersen, nilaiDp,
                    tenorTahun, tenorBulan, bunga, angsuran });
  },

  // Method 3: Update DOM dengan hasil
  tampilkan(d) {
    const el = document.getElementById('resultContainer');
    el.innerHTML = `<div class="result-animate">...${formatRupiah(d.angsuran)}...</div>`;
  },

  // Method 4: Reset form & tampilan
  reset() {
    document.getElementById('formKredit').reset();
    // Hapus semua is-invalid
    // Kembalikan resultContainer ke tampilan kosong
  }
};
JavaScript

Event & DOM Manipulation

🔄 Bagaimana JS Intercept Form PHP

Pola Dual Engine: Form aslinya dikirim ke PHP (method POST). JS "mencegat" submit event dengan e.preventDefault() dan menghitung sendiri — hasil tampil tanpa reload halaman!
JavaScript — DOMContentLoaded 
document.addEventListener('DOMContentLoaded', () => {
  initHargaFormat();  // Format input angka
  initBackToTop();   // Tombol scroll atas
  initNav();         // Smooth scroll
  initScrollSpy();   // Highlight nav aktif
  initKontak();      // Form kontak

  // Intercept form kredit → pakai JS, bukan PHP reload
  const form = document.getElementById('formKredit');
  if (form) {
    form.addEventListener('submit', e => {
      e.preventDefault(); // ← CEGAH reload halaman
      Kalkulator.hitung(); // ← Hitung di browser
    });
  }
});

👁 ScrollSpy — Deteksi Section Aktif

JavaScript — initScrollSpy 
function initScrollSpy() {
  const sections = document.querySelectorAll('section[id]');
  const links    = document.querySelectorAll('.nav-link');

  window.addEventListener('scroll', () => {
    let current = '';

    // Cek posisi setiap section
    sections.forEach(s => {
      if (scrollY >= s.offsetTop - navHeight - 20)
        current = s.id;
    });

    // Toggle class 'active' di nav link yang sesuai
    links.forEach(l => {
      l.classList.toggle('active',
        l.getAttribute('href') === `#${current}`);
    });
  });
}
JavaScript

Format Rupiah & Input Mask

Dua fungsi kecil tapi penting untuk pengalaman pengguna yang nyaman.

JavaScript — Format & Parse 
// Tampilkan angka sebagai Rupiah
function formatRupiah(angka) {
  return 'Rp. ' + Math.round(angka).toLocaleString('id-ID');
  // Contoh: 3333333 → "Rp. 3.333.333"
}

// Ubah "250.000.000" kembali ke angka 250000000
function parseRupiah(str) {
  return parseFloat(str.replace(/[^\d]/g, '')) || 0;
}

// Input mask — auto titik saat user ketik
function initHargaFormat() {
  const el = document.getElementById('hargaMobil');

  el.addEventListener('input', function() {
    const raw = this.value.replace(/[^\d]/g, ''); // ambil digit saja
    this.value = raw
      ? parseInt(raw, 10).toLocaleString('id-ID') // format ulang
      : '';
  });
}
🧠 Ingat mudah: formatRupiah = angka → teks tampil. parseRupiah = teks tampil → angka hitung. Dua arah, seperti AC.

Alur

Alur Kerja Lengkap

Berikut urutan proses dari pertama kali halaman dibuka sampai hasil muncul di layar:

1
🌐 Browser minta index.php
PHP menjalankan require_once kalkulasi.php → variabel $result = null, $errors = [], $posted = false.
2
📄 PHP render HTML kosong
Karena belum ada POST, halaman tampil dengan form kosong dan placeholder "Isi formulir dan klik Hitung Angsuran".
3
⚡ JavaScript init
DOMContentLoaded trigger semua fungsi init: format input, back-to-top, smooth scroll, scrollspy, kontak form, dan intercept form kredit.
4
✍️ User isi form
Saat ketik di harga, initHargaFormat() otomatis tambahkan titik ribuan real-time.
5
🖱 User klik "Hitung Angsuran"
JS intercept dengan e.preventDefault() → panggil Kalkulator.hitung().
6
🛡 Validasi JS
Cek semua field. Kalau ada yang salah → tambahkan class is-invalid dan hentikan proses. Tidak ada server request.
7
🧮 Hitung di Browser
Semua rumus dieksekusi di JS. Hasilnya disimpan di variabel lokal (angsuran, bunga, dll).
8
🎨 Update DOM
Kalkulator.tampilkan() mengubah innerHTML dari #resultContainer dengan HTML hasil. CSS fadeUp animasikan munculnya.
🔀 Fallback PHP: Jika JavaScript dimatikan, form submit normal ke server → PHP di kalkulasi.php hitung, dan hasilnya di-render saat halaman reload. Tetap berfungsi!
